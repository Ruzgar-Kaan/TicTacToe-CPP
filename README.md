# TicTacToe-CPP

A simple terminal-based Tic Tac Toe game written in C++.  
You play as **X**, and the computer plays as **O**.  
The game includes win detection, draw handling, and a replay option.

**********************************************************************

## Features
- 3x3 game board
- Player vs Computer gameplay
- Win detection (rows, columns, diagonals)
- Draw detection
- Randomized computer moves
- Option to replay after each match
  
**********************************************************************

## How to Compile

Use g++:
  g++ tic_tac_toe.cpp -o tictactoe

## How to Run
  ./tictactoe
